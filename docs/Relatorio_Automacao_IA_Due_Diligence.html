Relatório: Automação com IA

Possibilidades de integração de Inteligência Artificial no Sistema Summo Quartile

Summo Quartile · Abril 2026 · Documento interno para discussão com sócios

1. Sumário Executivo

Este relatório descreve as possibilidades de automação com Inteligência Artificial (IA) no Sistema Integrado Summo Quartile, com foco na Due Diligence Ambiental e nos demais módulos da plataforma.

A IA pode ser integrada de forma incremental, começando por funcionalidades de alto valor e baixo custo, evoluindo para automações mais sofisticadas conforme a demanda. Toda a infraestrutura técnica já está preparada — a ativação depende apenas de uma chave de API da Anthropic (Claude).

Investimento estimado: ~US$ 15/mês para uso moderado (3-5 análises/dia). Escala conforme uso. Sem compromisso mínimo — paga só o que usar.

2. Due Diligence Ambiental — 5 Oportunidades de IA

O módulo de Due Diligence é o produto premium da Summo. Hoje funciona com 5 fases e avaliação manual de 1.967 requisitos. A IA pode automatizar significativamente cada fase.

2.1. Identificação Automática de Documentos (Fase 2)

Fase 2 — Documentos

O que faz

Quando o usuário faz upload de um PDF, a IA lê o conteúdo e identifica automaticamente qual documento do inventário ele corresponde (EIA, RIMA, PCA, PRAD, etc.).

Como funciona

  1. Usuário arrasta PDF para a área de upload
  2. Sistema extrai o texto (pymupdf — já implementado)
  3. IA analisa as primeiras páginas e classifica: "Este é um EIA (Estudo de Impacto Ambiental)"
  4. Sistema marca automaticamente o documento como "Apresentado" no checklist

Valor

Elimina o trabalho manual de classificar documentos. O consultor faz upload de todo o book de uma vez e o sistema organiza automaticamente.

Custo por uso

~US$ 0,01 por documento (apenas lê as primeiras páginas).

2.2. Avaliação Automática de Requisitos (Fase 3)

Maior impacto

O que faz

Para cada requisito de conformidade, a IA lê o documento correspondente e avalia se o requisito é atendido, parcialmente atendido ou não atendido. Apresenta justificativa e trecho relevante do documento.

Como funciona

  1. Sistema seleciona o requisito: "O EIA contém análise de alternativas tecnológicas e locacionais?"
  2. IA busca no texto do EIA por trechos que respondam a essa pergunta
  3. Retorna: "Atende Parcialmente — O documento apresenta alternativas locacionais no capítulo 3.2, mas não discute alternativas tecnológicas."
  4. Radio button é pré-preenchido com badge "Sugerido por IA"
  5. Consultor aceita ou sobrescreve com um clique

Valor

Reduz o tempo de avaliação de horas para minutos. Um EIA com 843 requisitos que levaria dias para avaliar manualmente pode ser pré-analisado em minutos. O consultor revisa e valida, em vez de preencher do zero.

Custo por uso

~US$ 0,05-0,50 por documento (depende do tamanho do texto e número de requisitos). Uma DD completa com 10 documentos: ~US$ 1-5.

2.3. Geração de Plano de Ação (Fase 5)

Fase 5 — Resultado

O que faz

Para cada item "Não Atende", a IA gera recomendações detalhadas e contextualizadas de como corrigir a não-conformidade, incluindo referência à legislação aplicável e exemplos práticos.

Como funciona

  1. Sistema identifica os itens "Não Atende" e "Atende Parcialmente"
  2. IA recebe o contexto: requisito + legislação + trecho do documento (se houver)
  3. Gera recomendação: "Incluir no EIA seção dedicada a alternativas tecnológicas, conforme Art. 29, I da Lei 15.190/2025. Recomenda-se apresentar pelo menos 3 alternativas com análise comparativa de viabilidade técnica e ambiental."

Valor

Transforma o relatório de DD em um documento acionável. O cliente recebe não apenas o diagnóstico, mas o caminho para resolver cada problema. Aumenta significativamente o valor percebido do serviço.

Custo por uso

~US$ 0,02-0,10 por recomendação.

2.4. Mapeamento Automático de Legislação (Fase 0 — Preparação)

Alto valor estratégico

O que faz

A IA identifica automaticamente a legislação aplicável para um tipo de licenciamento + atividade + estado, e gera os requisitos de conformidade correspondentes.

Como funciona

  1. Consultor informa: LAC1, atividade A-02 (mineração), classe 4, MG
  2. IA consulta a base pré-mapeada (DN COPAM 217, LGLA, CONAMA)
  3. Verifica se houve atualizações legislativas desde o último mapeamento
  4. Complementa com requisitos adicionais que a IA encontra na legislação
  5. Sinaliza requisitos que podem ter sido afetados por alterações recentes

Valor

A base de requisitos nunca fica desatualizada. Quando uma nova deliberação é publicada, a IA detecta e sugere atualizações. Elimina o risco de trabalhar com requisitos obsoletos.

Custo por uso

~US$ 0,10-0,30 por consulta (texto legislativo pode ser extenso).

2.5. Verificação de Atualizações Legislativas (Contínuo)

Automação contínua

O que faz

Periodicamente (semanal ou mensal), a IA verifica se houve publicações no Diário Oficial que afetem os requisitos mapeados — novas deliberações, alterações de normas, revogações.

Como funciona

  1. Sistema consulta fontes públicas de legislação (Diário Oficial, CONAMA, COPAM)
  2. IA compara com a base de requisitos existente
  3. Gera alerta: "DN COPAM 246/2022 alterou critérios de classificação que afetam 23 requisitos da base"
  4. Consultor valida e a base é atualizada

Valor

Posiciona a Summo como referência em conformidade regulatória atualizada. Diferencial competitivo significativo.

Custo

~US$ 1-5/mês (verificação periódica com baixo volume).

3. Outros Módulos — Possibilidades de IA

3.1. Monitoramento de Indicadores — Alertas Inteligentes

O que faz

A IA analisa os dados de CFEM, produção, comércio exterior e licenciamento e gera briefings executivos automáticos com os destaques do período.

Exemplos de output

  • "A arrecadação de CFEM em ferro caiu 15% no último trimestre vs. mesmo período do ano anterior, concentrada em URA Sul de Minas."
  • "Sigma Lithium aumentou produção em 40% — requerimentos de expansão identificados no Cadastro Mineiro."
  • "Taxa de aprovação para atividade A-05-02-0 em classe 5 atingiu mínimo histórico de 35%."

Valor

Transforma dados em inteligência acionável. O visitante do site recebe insights que não encontra em nenhum outro lugar.

3.2. Monitoramento de Projetos — Curadoria Assistida

O que faz

A IA monitora notícias e press releases das grandes mineradoras e sugere atualizações para a base de 31 projetos curados.

Como funciona

  1. Periodicamente, IA pesquisa por notícias de "Vale S11D", "Anglo Minas-Rio", etc.
  2. Identifica mudanças: novos investimentos, licenças concedidas, atrasos
  3. Gera sugestão: "Atualizar projeto Vale S11D — comissionamento confirmado para Q3 2026 (fonte: vale.com, 15/03/2026)"
  4. Equipe Summo valida e atualiza a base

Valor

Mantém a base de projetos sempre atualizada com mínimo esforço manual. Cada atualização tem fonte rastreável.

3.3. Consulta Empresa — Dossier Enriquecido

O que faz

Quando o usuário consulta uma empresa por CNPJ, a IA gera um resumo executivo cruzando todos os dados disponíveis: decisões, infrações, CFEM, processos ANM, concessões.

Exemplo de output

"A empresa X possui 23 processos de licenciamento em MG, dos quais 18 foram deferidos (taxa 78%, acima da média do setor). Possui 3 infrações IBAMA nos últimos 5 anos, todas quitadas. CFEM regular desde 2019. Recomendação: perfil de baixo risco regulatório para novas operações."

Valor

O consultor recebe em segundos uma análise que levaria horas para compilar manualmente. Ideal para due diligence de M&A e prospecção de investimentos.

3.4. Mineradora Modelo — Diagnóstico Automatizado

O que faz

O cliente responde um questionário sobre sua operação e a IA compara com o modelo ideal, gerando um relatório de gaps e um plano de implementação personalizado.

Como funciona

  1. Cliente responde 20-30 perguntas por setor (Planejamento, Operação, etc.)
  2. IA compara respostas com os KPIs ideais da Mineradora Modelo
  3. Gera relatório: "Seu setor de Manutenção está no nível 2 (Reativo). O modelo ideal é nível 4 (Preditivo). Gaps identificados: MTBF abaixo do target, sem sistema de monitoramento de condição."
  4. Proposta de trabalho gerada automaticamente

Valor

Escala a capacidade de diagnóstico da Summo. Pode atender dezenas de clientes em paralelo, gerando propostas de consultoria automaticamente.

3.5. Chat Inteligente — Assistente do Sistema

O que faz

Um chat integrado ao sistema onde o usuário pode fazer perguntas em linguagem natural sobre os dados.

Exemplos

  • "Quais concessões de ouro estão ativas no Pará?" → tabela filtrada
  • "Qual a taxa de aprovação para mineração classe 5 em URA Central?" → resposta direta com dados
  • "Resuma o perfil ambiental da Vale em MG" → briefing contextualizado

Valor

Torna o sistema acessível para usuários não-técnicos. O sócio que não sabe navegar por filtros pode simplesmente perguntar.

Nota

A infraestrutura de chat já existe no sistema (/api/chat/stream com SSE). Precisa apenas de ativação e refinamento do prompt de sistema.

6. Requisitos Técnicos

O que já está pronto

  • SDK Anthropic instalado — pacote anthropic v0.91.0 já no ambiente
  • Endpoint de upload — POST /api/due-diligence/upload extrai texto de PDF
  • Endpoint de análise — POST /api/due-diligence/analyze preparado para Claude
  • Chat SSE — /api/chat/stream com streaming já implementado
  • Briefing scheduler — geração periódica de briefings já no código

O que falta para ativar

  1. Criar conta na Anthropic — console.anthropic.com
  2. Gerar API key — começa com sk-ant-...
  3. Configurar no sistema — criar arquivo .env com ANTHROPIC_API_KEY=sk-ant-...
  4. Pronto — as funcionalidades "ligam" automaticamente quando a chave está presente

Não é necessário nenhum outro software, servidor, ou configuração. O sistema já detecta a presença da chave e ativa os recursos de IA automaticamente.

8. Conclusão

A integração de IA no Sistema Summo Quartile representa um salto qualitativo na entrega de valor para os clientes e na eficiência operacional da equipe. As funcionalidades de maior impacto (avaliação automática de requisitos e plano de ação) podem ser ativadas em menos de uma semana após a configuração da API key, com custo mensal estimado de US$ 15-35.

A abordagem recomendada é incremental: validar com um caso real (Herculano), ajustar a qualidade das sugestões, e expandir gradualmente para os demais módulos.

Toda a infraestrutura técnica já está implementada. A decisão é exclusivamente de negócio.
