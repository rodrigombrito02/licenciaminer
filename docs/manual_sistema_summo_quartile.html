1. Visao Geral do Sistema

O Sistema Integrado Summo Quartile e uma plataforma de inteligencia estrategica para mineracao que cruza 16+ fontes publicas oficiais (IBAMA, ANM, SEMAD, CFEM, BCB, Comex, ICMBio, FUNAI) num unico lugar.

IndicadorValor
Processos minerarios mapeados303.250 (27 estados)
Decisoes ambientais analisadas42.758 (MG)
Requisitos de DD mapeados2.212
Documentos de licenciamento133
Projetos de mineracao curados31 (com fontes)
Fontes publicas integradas16+
Modalidades de licenca11 (incl. LGLA)

Stack tecnica: Frontend: Next.js 16 (App Router) | Backend: FastAPI + DuckDB | Deploy: Vercel + Railway

3.5. Due Diligence — 5 Fases

Produto premium da Summo Ambiental. Avaliacao de conformidade documental para licenciamento ambiental em 5 fases (metodologia PDCA).

[PRINT: http://127.0.0.1:3004/due-diligence]Step 1 com stepper mostrando 5 fases + formulario de configuracao

Fase 1: Configuracao e Escopo

  • 11 tipos de licenca (7 MG + 4 federais LGLA: LAU, LAC, LAE, LOC)
  • Atividades A-01 a A-07 (DN COPAM 217/2017)
  • Classes 1-6, CNPJ opcional
  • Legislacao: DN COPAM 217 + Lei 15.190/2025 + CONAMA 237/97, 9/90, 10/90
  • 133 documentos + 2.212 requisitos mapeados

Fase 2: Diagnostico Documental

  • Upload drag-and-drop: arraste PDFs para upload automatico
  • Matching inteligente: associacao 1:1, hierarquica, sem ambiguidade
  • Painel de resumo: X inseridos, Y identificados, Z pendentes + detalhes expandiveis
  • Auto-status: docs matchados marcados como "Apresentado" com badge "Vinculado"
  • Aviso: docs marcados manualmente sem PDF mostram "Sem documento vinculado"
  • 3 categorias: Core (detalhado), Existencia (binario), Suporte (generico 1-3 reqs)
[PRINT: http://127.0.0.1:3004/due-diligence]Step 2 com upload area + painel de matching + checklist

Fase 3: Avaliacao e Criticidade

  • Avaliacao de requisitos: radio buttons (Atende/Parcial/Nao Atende/N.A.)
  • Pre-preenchimento: docs ausentes = todos requisitos como "Nao Atende" automaticamente
  • Live score: barra de conformidade atualizada em tempo real
  • Criticidade inline: quadrantes (Acao Imediata / Secundaria / Baixa Prioridade)
  • Aderencia por tema: barras horizontais por area (Meio Biotico, Recursos Hidricos, etc.)
  • Gargalos identificados: temas com menor aderencia destacados
[PRINT: http://127.0.0.1:3004/due-diligence]Step 3 com requisitos pre-preenchidos + criticidade

Fase 4: Plano de Acao PDCA

  • Tabela de acoes priorizadas: docs ausentes primeiro, depois criticos, depois parciais
  • Badges de prioridade (Doc Ausente / Alta / Media)
  • Botao "Baixar Planilha" (xlsx com 4 abas)

Fase 5: Resultado Final

  • Gauge de conformidade + classificacao (Alta aderencia → Nao conforme)
  • Tabs: Visao Geral, Por Documento, Recomendacoes, Contexto Historico
  • Download PDF
  • Botao "Iniciar nova avaliacao"

Proximos passos

  • Geracao automatica de relatorios HTML por fase (infraestrutura em construcao)
  • Analise por IA (Claude API — infraestrutura pronta, requer API key ~US$15/mes)
  • Geracao de xlsx com formatacao condicional (endpoint pronto)

Proximo passo: Avaliacao em 5 Dimensoes Planejado

Modelo de avaliacao multidimensional para prospectos minerarios, cruzando dados publicos:

⚡
Energia
Disponibilidade de rede eletrica, distancia de subestacao, custo estimado de conexao, fontes alternativas (solar, eolica)
💧
Agua
Outorgas existentes (ANA/IGAM), bacia hidrografica, disponibilidade hidrica, conflitos de uso, qualidade
🏔
Cavidade
Presenca de cavernas/espeleologia (CECAV/ICMBio), restricoes legais, grau de relevancia, buffer de protecao
🚚
Logistica
Distancia ferrovia/porto/rodovia, modais disponiveis, custo de escoamento, capacidade instalada, gargalos
👥
Gente
Mao de obra (IDH, pop. urbana), comunidades no entorno, quilombolas, TIs, historico de conflitos, licenca social

Como funcionaria

  • Cada dimensao recebe score de 0-100 baseado em dados publicos cruzados
  • Score composto ponderado (radar chart / spider chart)
  • Identificacao automatica de restricoes criticas (ex: cavidade de maxima relevancia = veto)
  • Comparacao entre prospectos (qual area tem melhor perfil geral)
  • Dados: ANA (agua), ANEEL (energia), CECAV (cavidades), DNIT/ANTT (logistica), IBGE/FUNAI (gente)
