🌐Vercel

Frontend Next.js 16 + middleware Supabase pra proteção de rotas + Tailwind CSS.

  • Deploy automático a cada push
  • SSR + ISR + CDN global
  • Custo: R$ 0 (Hobby tier)

🚂Railway

Backend Python (FastAPI 0.135) + DuckDB + 16 fontes em Parquet + 5 SQLites isolados.

  • Build automático via Nixpacks
  • Volume persistente p/ parquets
  • Custo: ~R$ 0 (trial USD 5/mês)

🔐Supabase

Auth (email + senha + reset) com 4 níveis de role armazenados em user_metadata.

  • 50.000 MAU grátis
  • Admin API pra gerenciar usuários
  • Custo: R$ 0
Custo operacional atual: R$ 0/mês Tudo nos free tiers (Vercel Hobby + Railway trial + Supabase free). Quando exceder trial Railway (~5USD/mês), passa pra ~R$ 40/mês.

✓ Sprint 1 — Fundação

Sistema de 4 Roles + Home redesenhada + capa /ambiental

Auth Supabase com 4 níveis (visitante_free, visitante_pago, consultor, admin). useRole hook + RoleGate componente. Middleware protege rotas conforme role. Home com 3 layouts adaptativos. Capa /ambiental com 3 botões grandes.

roles middleware home redesign /ambiental
✓ Sprint 2 — Mineral Intelligence público

Capa de mercado + 2 abas novas (Ranking + Premium)

Hero chamativo com highlights ao vivo (Top município CFEM, top substância). Tab Ranking com Top 15 municípios e substâncias (barras coloridas). Tab Premium com 4 features gated por role. Conteúdo focado em atrair leads.

mercado público CFEM ranking Premium gated
✓ Sprint 3 — Sidebar minimalista + capas coerentes

De 42+ itens para 12 itens visíveis na sidebar

Capa /ferramentas-internas geral + capas /direitos, /sq-solutions, /riscos com mesmo padrão (hero + botões grandes + value props). Componentes compartilhados <ModuleHero> e <BigActionCard>. Remoção de menções hardcoded MUSA.

coerência sistêmica sidebar limpa capas standalone
✓ Sprint 4 — Funil de Oportunidades

Pipeline visual + avaliação 9 parâmetros + Relatório Viabilidade

Módulo isolado oportunidades.db. Kanban de 8 etapas (Prospect → Operação). Cada oportunidade tem avaliação de 9 parâmetros (água, energia, logística, mão de obra, licenciamento, financeiro, stakeholder, geológico, climático). Score consolidado automático. Geração de Relatório de Viabilidade Summo em HTML.

funil 8 etapas 9 parâmetros relatório HTML admin only
✓ Sprint 5 — Kanban + ClickUp/Trello CSV import

Visão Kanban no Plano de Ações + parser CSV

Tab "Kanban" adicionada ao detalhe do plano (com Gantt, EAP, Tabela). Auto-mapeamento de status text em buckets (A Fazer / Em Andamento / Bloqueado / Concluído / Cancelado). Parser CSV com auto-detect de delimitador e encoding. Upload aceita XLSX, XLSM, CSV, TSV — funciona com export ClickUp, Trello, Asana.

kanban CSV import ClickUp/Trello
✓ Sprint 6 — Riscos v2 (Cliente → Projeto)

Módulo de Riscos genérico multi-cliente

Backend completo riscos_v2.db isolado. Estrutura Cliente → Projeto (tipo erm/projeto/crise) → Risco. Matriz 5×5 com classificação automática (baixo/moderado/alto/crítico). 13 endpoints CRUD + matriz. Permite criar quantos clientes e projetos quiser. Frontend visual vem na próxima sprint.

multi-cliente matriz 5x5 13 endpoints
✓ Sprint 7 — Admin Dashboard + tracking

Painel admin com tráfego, conversões e gestão de usuários

Página /admin (visível só para role=admin). Pageview tracker automático no layout do dashboard. KPIs: eventos, usuários únicos, pageviews, CTAs. Filtro 7/30/90 dias. Top 15 páginas. Tráfego por role. Tab Usuários com listagem via Supabase Admin API.

tracking KPIs usuários Supabase
